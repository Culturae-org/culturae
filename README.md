# Culturae - Multi-Client Gaming Platform

> [!IMPORTANT]  
> Platform development and documentation are currently in progress.

A **centralized gaming platform for real-time multiplayer knowledge games**. Deploy one server, connect multiple clients (web, mobile, desktop, or custom).

## Culturae Platform

Culturae is a **multi-client gaming platform** where:
- **One central server** hosts all game logic and manages content
- **Multiple client implementations** (web, mobile, desktop) connect to the server
- **Different game modes**: Solo, 1v1, and Multi
- **Easy content management** via integrated admin dashboard
- **Cultpedia integration** for curated, multilingual question and geography datasets
